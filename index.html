Перевод

Чтобы перевести деньги: откройте карту счета, нажмите «Перевести», введите номер телефона и сумму.

1. Зайдите в карту счета
2. Нажмите «Перевести»
3. Введите номер телефона
4. Укажите сумму
5. Нажмите «Перевести»
Установка пин-кода

Пин-код позволяет быстро входить в приложение без ввода пароля.

1. Нажмите на аватар вверху экрана
2. Введите новый пин-код (4 цифры)
3. Подтвердите пин-код
4. Нажмите «Сохранить»
5. Включите Face ID при желании
